# General AI Assistant

General-purpose AI Assistant

## Overview

A versatile AI assistant with file operations and shell command capabilities. Suitable for research, writing, code analysis, and everyday tasks. Powered by OpenRouter for flexible model selection.

## Capabilities

- **File operations** — Read, write, and manage project files
- **Shell commands** — Execute bash commands for automation
- **Research & writing** — Analyze documents, write reports, summarize content
- **Code analysis** — Read and understand codebases, suggest improvements
- **Curated Skills** — Access to vetted, localized skill definitions
- **Community Skills** — Access to SkillsMP marketplace for 100+ additional skills (when connected)

## Use Cases

- "Summarize these meeting notes" — Extract key decisions and action items
- "Analyze this CSV and generate a report" — Data processing with insights
- "Search the codebase for all API endpoints" — Code exploration and documentation

## Technical Details

- **Agent Family**: OMC Talent
- **Hosting**: Company-hosted
- **Tools**: Filesystem, Bash, Curated Skills, Community Skills (optional)
